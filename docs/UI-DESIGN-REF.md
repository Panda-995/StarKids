
### 1.2 色彩使用规范

| 元素 | 颜色 | 说明 |
|---|---|---|
| 主按钮 | `bg-candy-blue` | 小朋友端主要操作 |
| 次按钮 | `bg-candy-green` | 次要操作 |
| 积分相关 | `bg-brand-500` | 星星/积分元素 |
| 成功 | `bg-candy-green` | 审核通过/完成 |
| 警告 | `bg-candy-orange` | 待审核/注意 |
| 错误 | `bg-candy-red` | 审核拒绝/错误 |
| 背景 | `bg-warm-50` | 页面底色 |
| 卡片 | `bg-white shadow-soft` | 内容卡片 |
| 文字主色 | `text-warm-700` | 标题/正文 |
| 文字次色 | `text-warm-400` | 辅助说明 |

## 二、小朋友端页面设计

### 2.1 字体大小规范 (小朋友端 — 更大！)

| 用途 | 大小 | 行高 | 字重 |
|---|---|---|---|
| 页面标题 | 28px | 1.4 | 600 |
| 卡片标题 | 22px | 1.4 | 600 |
| 正文 | 18px | 1.6 | 400 |
| 辅助文字 | 16px | 1.5 | 400 |
| 大数字 (积分) | 36px | 1.2 | 700 |
| 按钮文字 | 18px | 1.4 | 600 |

> 小朋友端所有字体比常规大 20-30%，方便小朋友阅读

### 2.2 按钮尺寸规范

| 类型 | 高度 | 圆角 | 内边距 |
|---|---|---|---|
| 主按钮 | 56px | 28px | 20px 24px |
| 次按钮 | 48px | 24px | 16px 20px |
| 图标按钮 | 56px | 28px | — |
| 完成任务按钮 | 64px | 32px | 24px 32px |

> 小朋友的按钮要 **大**！方便小手点击

### 2.4 任务完成动画流程

```
小朋友点击 "我做完啦！"
        ↓
   按钮弹跳 + 震动
        ↓
   🎊 屏幕撒花 (Confetti 效果)
        ↓
   ⭐ 星星从任务卡片飞出
        ↓
   飞入顶部积分罐
        ↓
   💬 弹出气泡: "太棒啦！+15 积分！"
        ↓
   🐱 宠物跳出来: "恭喜恭喜！"
        ↓
   返回任务列表 (任务标记 ⏳ 待审核)
        ↓
   (家长批准后) → 再次撒花 → 积分真正到账
```

## 三、家长端页面设计

### 3.1 家长端风格

家长端风格与小朋友端区分开——**简洁专业**但不失温暖：

| 属性 | 值 |
|---|---|
| 主色 | `#4A90D9` (沉稳蓝) |
| 背景 | `#F8F9FB` (浅灰蓝) |
| 卡片 | 白色 + 柔和阴影 |
| 字体 | Noto Sans SC (简洁现代) |
| 圆角 | 12px (比小朋友端小) |
| 字号 | 常规 (14-16px) |

## 四、插画与素材建议

### 4.1 图标风格

| 元素 | 推荐风格 |
|---|---|
| 导航图标 | Line Awesome / Lucide (加粗描边) |
| 任务图标 | Emoji 风格 (大尺寸) |
| 成就徽章 | 圆形徽章 + 渐变填充 |
| 分类图标 | 扁平插画风 |
| 宠物 | Lottie 动画 (可爱风) |

### 4.2 插画来源

| 素材 | 推荐来源 |
|---|---|
| 儿童插画 | Storyset (儿童主题) |
| 卡通人物 | Freepik (Kids Illustration) |
| 图标集 | Iconoir / Lucide |
| Lottie动画 | LottieFiles (搜索 "kids", "cute") |
| 音效 | FreeSound (搜索 "cartoon", "cute", "success") |
| 粒子效果 | canvas-confetti / tsParticles |

### 4.3 宠物动画状态

| 状态 | 动画 |
|---|---|
| 蛋宝宝 | 轻轻摇晃 |
| 开心 | 跳跃 + 闪光 |
| 难过 | 趴下 + 掉眼泪 |
| 吃饭 | 咀嚼动画 |
| 进化 | 光效变身 |
| 睡觉 | 打呼噜 (Zzz) |

## 五、暗黑模式 (家长端)

家长端支持暗黑模式，小朋友端**不支持**（保护视力，保持明亮）：

```css
/* 暗黑模式配色 */
--bg-primary: #1A1D23;
--bg-secondary: #252830;
--bg-card: #2A2D35;
--text-primary: #E8E8E8;
--text-secondary: #A0A0A8;
--border: #3A3D45;
--brand: #5BA0F0;
```
